Perguntas frequentes

Como faço para instalar o Chrome?

Para instalar o Chrome, basta fazer o download do arquivo de instalação e procurar por ele na sua pasta de downloads. Abra o arquivo e siga as instruções. Depois que o Chrome estiver instalado, você poderá excluir o arquivo de instalação. Saiba mais sobre como fazer o download do Chrome

O Chrome funciona no meu sistema operacional?

O Chrome é compatível com dispositivos que executam os sistemas operacionais Windows e Mac, desde que eles atendam aos requisitos mínimos do sistema. Para instalar o Chrome e receber o suporte adequado, é necessário atender aos requisitos do sistema. Saiba mais sobre o uso do Chrome no seu dispositivo.

Como faço para tornar o Google Chrome meu navegador padrão?

Você pode definir o Chrome como o navegador padrão nos sistemas operacionais Windows e Mac e no seu iPhone, iPad ou dispositivo Android. Depois de definir o Chrome como o navegador padrão, todos os links em que você clicar serão abertos automaticamente nesse navegador. Para instruções específicas do seu aparelho, consulte este link.

Quais são as configurações de segurança do Chrome?

O Chrome conta com recursos modernos de segurança e proteção para que você possa ficar no controle. Use a Confirmação de segurança para conferir se há senhas comprometidas, verificar o status da Navegação segura e procurar atualizações disponíveis. Saiba mais sobre a segurança e proteção no Chrome.
